Edição

A seleção de texto permite executar diversas operações de edição, como:

Selecione uma palavra ou trecho na caixa de texto abaixo e use CTRL + C para copiar o trecho selecionado para a área de transferência. Depois, cole o trecho copiado no campo 1 do formulário com CTRL + V.

Selecione uma palavra ou trecho na caixa de texto abaixo e use CTRL + X para recortar o trecho selecionado para a área de transferência. Depois, cole-o no campo 2 do formulário com CTRL + V.

Selecione uma palavra ou trecho na caixa de texto abaixo e use CTRL + C para copiar o trecho selecionado para a área de transferência. Depois, selecione uma palavra no campo 1 do formulário e pressione CTRL + V.

Selecione uma palavra ou trecho no campo 2 do formulário e pressione a tecla de espaços.

Dicas:

  • Para selecionar todo o texto na caixa de texto, use as teclas Ctrl + A.
  • Para desfazer uma ação de edição, use as teclas Ctrl + Z.
  • Para refazer uma ação de edição desfeita, use as teclas Ctrl + Y.
