Edição

A seleção de texto permite executar diversas operações de edição, como:

  • CTRL +C (Copiar): duplica o trecho de texto selecionado na área de transferência.
  • CTRL +X (Recortar): remove o trecho de texto selecionado para a área de transferência.
  • CTRL +V (Colar): insere o conteúdo da área de transferência no local atual do cursor.
  • DEL ou BACK SPACE: exclui o trecho de texto selecionado.
  • Substitua o texto selecionado por um ESPAÇO ou com o atual conteúdo da área de trasferência

Dicas:

Para selecionar todo o texto na caixa de texto, use as teclas Ctrl + A.

Para desfazer uma ação de edição, use as teclas Ctrl + Z.

Para refazer uma ação de edição desfeita, use as teclas Ctrl + Y.
