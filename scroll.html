Scroll do Mouse

O scroll do mouse é usado para rolar uma página para cima ou para baixo. Para rolar para cima, basta girar a roda do mouse na direção contrária à do usuário. Para rolar para baixo, basta girar a roda do mouse na direção do usuário.
